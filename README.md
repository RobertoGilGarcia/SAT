# ASIGNATURA SERVICIOS Y APLICACIONES TELEMÁTICAS 2025-2026

Este es el repositorio correspondiente a la asignatura Servicios y Aplicaciones Telemáticas, donde se almacenan las diferentes prácticas que se han realizado, incluyendo el proyecto final

Esta asignatura se basa en la creación del backend de páginas web utilizando el lenguaje de programación **Python** y el framework  de desarrollo web **Django**

De forma básica se ve algo de frontend con CSS, HTML, HTMX y Bootstrap, pero esta asignatura no se centra en el frontend.

Al inicio de la asignatura, se desarrolla una introducción utilizando sockets en Python y gestionando respuestas en crudo, para poder comprender como funcionan las interacciones HTTP en una página web.
