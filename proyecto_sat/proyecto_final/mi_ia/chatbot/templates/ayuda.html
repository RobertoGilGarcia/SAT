¿Qué es ChatIA?

ChatIA es una aplicación web que simula un asistente conversacional. Permite escribir mensajes, recibir respuestas generadas por una inteligencia artificial y guardar el historial de cada conversación.

Cómo usar la aplicación

Para usar el chat necesitas crear una cuenta o iniciar sesión. Una vez dentro, podrás acceder a tu cuenta, crear conversaciones nuevas, escribir mensajes y consultar conversaciones anteriores.

Cada conversación queda guardada en la base de datos junto con los mensajes del usuario y las respuestas generadas por la IA.

Conversaciones

La aplicación permite gestionar varias conversaciones. Cada usuario ve únicamente sus propios chats. Desde la cuenta se pueden recuperar conversaciones antiguas, continuar escribiendo en ellas o borrarlas.

Perfil y configuración

Cada usuario dispone de una zona de configuración donde puede modificar algunos datos de su perfil, como el alias, tu correo electrónico o preferencias en el tema visual.
